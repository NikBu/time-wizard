# time-wizard
Small web-app for easier time and task management.

## Features

- **Timers**
  - Multiple concurrent timers with names and presets (Pomodoro, breaks, deep work, etc.)
  - Rich repeat modes: once, fixed repeats, increasing/decreasing intervals, custom sequences, Fibonacci, random range
  - Built‑in synthesized alert sounds (bell, viola, harp, chime, drum, whistle)
  - Custom alarm sounds via file upload
  - Real‑time countdown and progress bars

- **Checklists**
  - Multiple lists (Work, Home, Learning, etc.)
  - Hierarchical tasks (parent + subtasks)
  - Per‑task point values and list progress
  - “Clear done” and list duplication

- **Themes & Appearance**
  - Several visual themes (Old Parchment, Dark Sanctum, Enchanted Forest, Ember & Ash, Arctic Scholar)
  - Light/dark mode toggle
  - Smooth typography and layout tuned for productivity use

- **Archibald Companion**
  - Owl librarian assistant with multiple moods and sprites
  - Reacts to timers, task completion, long sessions, and time of day
  - Floating popup quotes across all tabs
  - FAQ section explaining features in‑character

- **Import / Export**
  - Export timers, checklists, points, and settings to a JSON file
  - Import that JSON back to restore your state
  - Custom audio files are *not* included (binary)
